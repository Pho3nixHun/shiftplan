# JANE-kickstart
**J**spm (with systemjs) + **A**ngular + **N**ode + **E**xpress

Included libraries:

### Server
    
+ Composed on Node 7.0.0
+ Expressjs
+ Death
+ Winston-express
+ body-parser
+ lets-encrypt-express
+ compression
    
### Client

+ Angular
+ Angular-animate
+ Angular-messages
+ Angular-resource
+ Angular-route
+ Angular-material
+ Angular-formly
+ Angular-formly-material
+ Api-check
+ Animate.css
+ Material-icons font set

Systemjs-builder also implemented. See ```builder.js```.
